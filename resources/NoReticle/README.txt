/-/ INSTALLATION \-\
1. Drag and drop the "NoReticle" folder into your server's resource folder.
2. Add "start NoReticle" to your server config to start the script when your server starts.

/-/ PERMISSIONS \-\
Add the following line to your permissions config:
add_ace group.admin "Reticle" allow

The above permission makes the reticle visible to anyone with the group of admin.
You can change the group to moderators or any other groups you have setup accordingly.
Permissions setup support will be limited.
